# Softbear

Desarrollaremos un sistema para el Festival de las Luciernagas como parte de nuestro proyecto final de ingenieria de Software con ayuda del framework de trabajo Django

## Instalación

1. Creamos un entorno virtual tomando los requerimientos del archivo environment.yml 
(Lo descargamos del repositorio)

$ conda env create -f environment.yml

2. Activamos nuestro environment de conda con

$ conda activate Sotfbear

3. Despues de crear nuestro entorno virtual clonamos el repositorio

## Uso

Estaremos trabajando en diversas ramas para llevar la correcta organizacion del proyecto
